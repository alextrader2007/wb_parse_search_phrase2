# Парсер товаров Wildberries на Python

Этот скрипт представляет собой гибридный парсер товаров маркетплейса Wildberries. Он собирает как базовую, так и расширенную информацию о товарах напрямую через внутренние API Wildberries, используемые на сайте, что обеспечивает высокую скорость работы и устойчивость к блокировкам.

## Возможности парсера
- **Два режима работы**: 
  - По списку артикулов (SKU).
  - По ключевому слову (поисковой строке) с возможностью ограничения количества страниц.
- **Сбор подробной информации**:
  - *Базовая*: Артикул, Название, Бренд, Цена без скидки, Цена со скидкой (СПП), Рейтинг, Количество отзывов.
  - *Расширенная*: Информация о продавце (поставщике), размеры товара, ссылка на изображение, детальные остатки по складам.
- **Поддержка прокси**: Возможность загрузки списка прокси-серверов из файла `proxies.txt` для ротации и обхода лимитов/блокировок.
- **Декодирование складов**: Автоматический сбор и использование официального справочника складов Wildberries для отображения человекопонятных названий (например, «Коледино», «Казань», «Электросталь») вместо числовых ID.
- **Удобный экспорт**: Сохранение результатов в форматах Excel (`.xlsx`) и CSV (с поддержкой UTF-8-sig для корректного открытия в Excel на русском языке).
- **Красивый консольный интерфейс**: Использование библиотеки `rich` для отображения прогресс-баров, интерактивных меню выбора и красивых таблиц с превью результатов прямо в терминале.

## Установка и запуск

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python (версии 3.8 и выше). Установите необходимые библиотеки, выполнив команду в терминале:

```bash
pip install -r requirements.txt
```

### 2. Подготовка прокси (опционально)
Если вы планируете собирать большое количество товаров или часто запускать парсер, создайте в папке проекта файл `proxies.txt` и добавьте туда ваши прокси (по одному адресу на строку). Поддерживаются форматы:
```text
http://user:password@ip:port
ip:port
```
Скрипт автоматически обнаружит и будет использовать их по очереди.

### 3. Запуск парсера

#### Интерактивный запуск (скрипт сам спросит все параметры в консоли):
```bash
python wb_parser.py
```

#### Запуск с аргументами командной строки:

**Режим сбора по артикулам:**
```bash
python wb_parser.py -m sku -s "7403217, 13859082, 172948723" -o my_skus
```

**Режим сбора по поисковому запросу:**
```bash
python wb_parser.py -m search -q "кроссовки мужские" -p 3 -o search_results
```
*Где:*
*   `-m` / `--mode` — режим работы (`sku` или `search`)
*   `-s` / `--skus` — перечень артикулов через запятую или пробел
*   `-q` / `--query` — поисковый запрос в кавычках
*   `-p` / `--pages` — количество страниц для парсинга (по умолчанию 5)
*   `-o` / `--output` — базовое название файлов результатов (будут созданы `<output>.xlsx` и `<output>.csv`)
*   `--proxy-file` — путь к вашему файлу прокси (если имя отличается от `proxies.txt`)

## Важные примечания
1. **Региональные цены**: Скрипт использует код региона Москвы (`dest=-1257786`) по умолчанию. Цены и остатки могут отличаться для других регионов (например, Дальнего Востока или Сибири).
2. **Лимиты частоты запросов**: Во избежание бана вашего IP/прокси, в коде настроены случайные задержки между запросами. При сборе очень большого количества данных рекомендуется использовать качественные ротируемые прокси.
